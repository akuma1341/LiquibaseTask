Задача 4.1, работа с liquibase и h2
1. Создать новый репозиторий, spring boot gradle project (https://start.spring.io), добавить условие в readme.md, закоммитить структуру в мастер, создать ветку liquibase-task.
2. Изучить библиотеку liquibase и h2 db, что это такое и для чего, подключить ее к проекту.
3. Создать новую схему данных в mysql, создать ченжсеты для создания таблиц из задачи 3, для написания ченжсетов использовать xml.
4. Добавить ченжсет в отдельном файле для наполнения таблиц данными, используя тег sql.
5. Добавить классы ентитней и классы репозиториев.
6. Добавить API для получения количества строк в каждой таблицы.
7. Добавить простые тесты на апи, убедиться, что h2 работает (добавить тест, в котором получить количество строк, добавить запись через репозиторий, получить количество строк, убедиться, что количество изменилось на 1, проверить mysql, новой записи не должно быть).
8. Скинуть ссылку на пул реквест в чат для проверки.
   
Цель: изучить основы liquibase.
